# CCNA Lab 01 - OSPF Redundancy

## Overview

This lab was created using Cisco Packet Tracer to practice routing concepts and network redundancy.

## Objectives

- Configure multiple LANs
- Connect routers using serial links
- Implement OSPF Area 0
- Replace static routes with dynamic routing
- Test redundant paths and failover scenarios

## Topology

- 4 Cisco routers
- Multiple LAN segments
- Redundant connections between routers
- OSPF as routing protocol

## Skills Practiced

- IP addressing
- Subnetting
- Static Routing
- OSPF Configuration
- Troubleshooting
- Network Redundancy
